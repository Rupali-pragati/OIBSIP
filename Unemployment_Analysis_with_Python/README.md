# Unemployment Analysis with Python

## Overview
This project analyzes unemployment trends during the COVID-19 period using Python. It includes exploratory data analysis (EDA), visualizations, and insights on how unemployment varied across Indian regions.

## Objectives
- Analyze unemployment rate data
- Visualize trends and regional variations
- Identify the impact of COVID-19 on employment

## Dataset
Dataset: [Unemployment in India - Kaggle](https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india)  
Downloaded automatically using `kagglehub`.

## Steps Performed
1. Imported and cleaned dataset  
2. Handled missing values and formatted dates  
3. Created visualizations:
   - Average Unemployment by Region
   - Unemployment Trends Over Time
   - State-wise Distribution (Boxplot)
   - Correlation Heatmap  
4. Derived insights from data

## Results
- Significant rise in unemployment observed during lockdown periods.
- Regional differences highlight varying economic recovery rates.
