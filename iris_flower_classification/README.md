# OIBSIP
# Iris Flower Classification (Machine Learning Project)

This project classifies Iris flowers into three species — *Setosa*, *Versicolor*, and *Virginica* — based on their sepal and petal measurements using Machine Learning.


### Features
 Loads dataset directly from KaggleHub
 Data analysis and visualization
 Machine learning model training
 Model comparison (Logistic Regression, Decision Tree, Random Forest)
 Accuracy evaluation
 Sample prediction demo

### Technologies Used
 Python
 Scikit-learn
 Pandas, NumPy
 Matplotlib, Seaborn
